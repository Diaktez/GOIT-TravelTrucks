Web application for renting campers inside the "TravelTrucks" company. View our
fleet, filter by your needs, and book your dream van for the next adventure.

## Features

- **Catalog:** Browse a comprehensive list of campers with infinite scroll (Load
  More).
- **Filtering:** Filter vehicles by location, equipment (AC, Kitchen, TV, etc.),
  and vehicle type.
- **Details Page:** View detailed specifications, gallery, and user reviews for
  each camper.
- **Favorites:** Save your favorite campers to a persistent list (LocalStorage).
- **Booking:** Simple booking form with validation.

## Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Styling:** CSS Modules
- **HTTP Client:** Axios
- **Icons:** SVG Sprites

## Installation & Run

Clone the repository:

git clone
[https://github.com/YOUR_USERNAME/travel-trucks.git](https://github.com/YOUR_USERNAME/travel-trucks.git)

Install dependencies:

    npm install

# or

    yarn install

Run the development server:

    npm run dev Open http://localhost:3000 with your browser.
